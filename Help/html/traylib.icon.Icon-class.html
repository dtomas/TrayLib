get_icon_names(self)

source code 

Override this to determine the icon's icon names. These will only be used if get_icon_path doesn't return a path to an icon or the icon could not be loaded.

@return The icon names.

get_icon_pixbuf(self)

source code 

Override this to determine the pixbuf to be used for the icon. This will only be used if no icons could be found for the icon names returned by get_icon_names() and get_icon_path() doesn't return a path to an icon.

make_icon(self)

source code 

This determines the gtk.gdk.Pixbuf the Icon should have. By default, this method tries to load the pixbuf from the path returned by get_icon_path(). If this fails, it looks up the icon in the current icon theme from the icon names returned by get_icon_names(). If this also fails, it returns the pixbuf returned by get_icon_pixbuf().

Returns:
The new pixbuf.
